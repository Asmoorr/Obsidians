Guitar Dashboard / Database - приложения для удобства организации гитарной информации: песен, табов, а также открытия файлов табов, песен, пресетов VFX

### 1. Структура данных (Карточка песни)

То, что описывает песню и позволяет быстро ориентироваться в списке.

- **Основная информация:** Название песни, Исполнитель, тональность (автоопределитель?). Дополнительно хранить состав группы для поиска по участникам
- **Строй:** Теги (Standard E, Drop C, DADGAD и т.д.)
- **Секция:** Что именно разбираем? (Ритм, Соло-1, Соло-2, Вся песня, Акустическая вставка).
- **Статус (Status):** Текущее состояние.
    - В планах
    - Учу
    - Повторить
    - Знаю
- **Технические параметры:**
    - **BPM:** Темп оригинала.
    - **Каподастр:** Номер лада (если нужен).
        
- **История:** 
	- Дата последнего занятия (автоматически обновляется при взаимодействии с карточкой)
	- Счетчик "Сколько раз играл".
    

### 2. Медиа и Файлы (Хранение и Доступ)

Централизация всего контента, чтобы не искать по папкам.

- **Ссылки на Табулатуры:**
    - Кнопка открытия видео-табы на YouTube.
    - Кнопка открытия страницы на Songsterr.
- **Работа с Guitar Pro (.gp / .gp5 / .gpx):**
    - Загрузка файла в карточку.
    - Кнопка **"Скачать"** (для переноса на другое устройство).
    - Кнопка **"Открыть"** (триггер запуска установленного Guitar Pro на ПК).
- **Работа с Аудио (MP3/WAV/Backing Track):**
    - Встроенный плеер (Play/Pause/Seek) прямо в карточке (для быстрого прослушивания).
    - Кнопка скачивания файла.
    - Кнопка **"Открыть в Transcribe!"** (Специфическая фича: запуск внешнего приложения с загруженным треком).
    - Возможность переключаться между оригиналом, минусом, (синтезом из табулатуры?)
    - Возможность транспонировать песню под строй
    - Зациклить / замедлить песню
- **Пресеты:**
    - Возможность прикрепить файл пресета (для Neural DSP, Helix, BIAS и т.д.)

### 4. Интерфейс и UX (Сценарии использования)

- **Dashboard (Главная):** Сетка или список карточек.
    
- **Умная фильтрация:**
    
    - "Показать только Drop C" (взял гитару в этом строе — играешь только эти песни).
    - "Показать то, что я учу сейчас" (фильтр по статусу).
    - Полнотекстовый поиск по названию / исполнителю / описанию? (если добавлю)
        
- **Быстрый доступ:** Кнопки действий (Play, Tab, GP) должны быть доступны сразу с фасада карточки, без необходимости "проваливаться" внутрь для каждого клика.

- Drag & Drop загрузка чего-либо: mp3, gp и тд в новую песни или добавление в старую

- **Random picker:**
    
    - Кнопка "Сыграть случайную песню" (для тех, кто не знает, что сегодня практиковать)
        
    - Фильтр для random picker (только "В планах" или только "Повторить")

- Возможность менять размер отображения карточек (сетка, построчно)
- Календарь практик (тепловая карта активности как на GitHub)
