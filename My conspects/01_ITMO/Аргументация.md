## Спор

**Спор** — процесс отстаивания **противоположных** (несовместимых) точек зрения.

### Участники спора

| Роль | Описание |
|---|---|
| **Пропонент** | Предлагает позицию (выдвигает **тезис**) |
| **Оппонент** | Отрицает истинность тезиса (выдвигает **антитезис**) |

> [!important] Бремя доказательства лежит на пропоненте

Антитезис может быть сформулирован очень по-разному.

### Виды спора

| Вид | Краткое описание |
|---|---|
| **Дискуссия** | Поиск истины через совместное обсуждение |
| **Диспут** | Публичный спор на заданную тему |
| **Дебаты** | Регламентированное столкновение позиций |
| **Полемика** | Спор ради победы, а не ради истины |

> [!quote] Истина рождается в дискуссии (одном из видов спора)

### Вопрос-ответные процедуры в споре

- **Риторические вопросы** — заставляют аудиторию согласиться с позицией
- **Уточняющие, проясняющие вопросы** — реальные вопросы для восполнения информации
- **Вопрос как «оттягивание доказательства»** — позволительная уловка, чтобы выиграть время
- **Вопросы с ложными пресуппозициями** — вопросы, содержащие скрытое ложное утверждение
- **Майевтика** — вопросы Сократа, которые подводят собеседника к нужному выводу

### Модель критической дискуссии

1. **Стадия конфронтации** — выявление разногласий
2. **Стадия открытия дискуссии** — распределение ролей и правил

## Уловки в споре

> [!warning] Уловки — приёмы, позволяющие получить преимущество в споре нечестным путём

### Логические уловки

| Уловка                                    | Суть                                                                                            |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------- |
| **Злостное отрицание доводов**            | Безосновательное отрицание аргументов оппонента                                                 |
| **Ложный отвод довода**                   | Утверждение, что довод не относится к предмету спора                                            |
| **Расчёт на медленность мышления**        | Быстрая подача сложных аргументов, рассчитанная на доверчивость                                 |
| **Ставка на ложный стыд**                 | «Общеизвестно, что...», «Вы не станете отрицать, что...» — давление авторитетом «общего знания» |
| **Перекладывание бремени доказательства** | Требование от оппонента доказать обратное вместо обоснования своего тезиса                      |

### Психологические уловки

| Уловка                       | Суть                                                                                                  |
| ---------------------------- | ----------------------------------------------------------------------------------------------------- |
| **«Механическая»**           | Не давать противнику говорить, перебивать                                                             |
| **Выведение из равновесия**  | Раздражить противника и вывести из себя                                                               |
| **«Подмазыванье» аргумента** | Лесть или излишнее уважение, чтобы противник согласился                                               |
| **Внушение**                 | Громкий голос, уверенный тон, авторитетные манеры — всё это даёт огромное преимущество в устном споре |
| **Повторение**               | Многократное повторение одного и того же довода для закрепления в сознании                            |
| **Риторический вопрос**      | Вопрос, не требующий ответа, заставляет аудиторию согласиться                                         |

### Тактические уловки

| Уловка                              | Суть                                                          |
| ----------------------------------- | ------------------------------------------------------------- |
| **Оттягивание возражения**          | Применяется, когда нужный довод не приходит в голову сразу    |
| **«Напирать» на довод**             | Давление на аргумент, который неудобен противнику             |
| **«Призыв» / «довод к городовому»** | Апелляция к авторитету или силе вместо аргументов             |
| **«Чтение в сердцах»**              | Разбор не слов оппонента, а его предполагаемых тайных мотивов |
