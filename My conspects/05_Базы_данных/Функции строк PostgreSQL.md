Для каждой функции приведено по 3 примера:
**Б** — Базовый
**Т** — Тематический (реальный)
**Г** — Граничный (edge case)

### 🔍 Часть 1. Поиск, извлечение и анализ
Функции для того, чтобы найти нужный фрагмент в тексте, откусить его часть или узнать размер.

| Функция        | Описание                                                  |           Тип           | Пример выражения                                                                                               | Результат                                  |
| :------------- | :-------------------------------------------------------- | :---------------------: | :------------------------------------------------------------------------------------------------------------- | :----------------------------------------- |
| **LEFT**       | Возвращает первые *n* символов строки слева.              | **Б**<br>**Т**<br>**Г** | `LEFT('ABC', 1)`<br>`LEFT('Университет ИТМО', 11)`<br>`LEFT('Кафка', -2)`*                                     | `'A'`<br>`'Университет'`<br>`'Каф'`        |
| **RIGHT**      | Возвращает последние *n* символов справа.                 | **Б**<br>**Т**<br>**Г** | `RIGHT('ABC', 2)`<br>`RIGHT('Три товарища', 8)`<br>`RIGHT('Fender Stratocaster', -7)`                          | `'BC'`<br>`'товарища'`<br>`'Stratocaster'` |
| **SUBSTRING**  | Извлекает подстроку начиная с указанной позиции.          | **Б**<br>**Т**<br>**Г** | `SUBSTRING('ABC', 1, 1)`<br>`SUBSTRING('Франц Кафка: Процесс', 14, 7)`<br>`SUBSTRING('ИТМО', 10, 5)`           | `'A'`<br>`'Процесс'`<br>`''` *(пусто)*     |
| **SPLIT_PART** | Разбивает строку по разделителю и возвращает *n*-ю часть. | **Б**<br>**Т**<br>**Г** | `SPLIT_PART('A-B-C', '-', 2)`<br>`SPLIT_PART('Fender,Gibson,Ibanez', ',', 2)`<br>`SPLIT_PART('Кафка', '-', 2)` | `'B'`<br>`'Gibson'`<br>`''`                |
| **POSITION**   | Возвращает позицию начала подстроки.                      | **Б**<br>**Т**<br>**Г** | `POSITION('B' IN 'ABC')`<br>`POSITION('Мария' IN 'Эрих Мария Ремарк')`<br>`POSITION('x' IN 'ИТМО')`            | `2`<br>`6`<br>`0`                          |
| **LENGTH**     | Возвращает количество символов в строке.                  | **Б**<br>**Т**<br>**Г** | `LENGTH('ABC')`<br>`LENGTH('Жизнь взаймы')`<br>`LENGTH(NULL)`                                                  | `3`<br>`12`<br>`NULL`                      |

*\* В PostgreSQL отрицательное значение в `LEFT/RIGHT` отбрасывает символы с противоположного конца.*

### ✂️ Часть 2. Модификация, замена и регулярки
Функции для переписывания истории, исправления ошибок и сложного поиска по шаблонам.

| Функция | Описание | Тип | Пример выражения | Результат |
| :--- | :--- | :---: | :--- | :--- |
| **REPLACE** | Заменяет все вхождения подстроки на новую. | **Б**<br>**Т**<br>**Г** | `REPLACE('ABC', 'B', 'A')`<br>`REPLACE('Акустика', 'Акустика', 'Электрогитара')`<br>`REPLACE('Кафка', 'а', '')` | `'AAC'`<br>`'Электрогитара'`<br>`'Кфк'` |
| **REVERSE** | Переворачивает строку задом наперед. | **Б**<br>**Т**<br>**Г** | `REVERSE('ABC')`<br>`REVERSE('Замок')`<br>`REVERSE('')` | `'CBA'`<br>`'комаЗ'`<br>`''` |
| **REGEXP_MATCHES** | Ищет совпадения по POSIX регулярному выражению. | **Б**<br>**Т**<br>**Г** | `REGEXP_MATCHES('ABC', '^(A)(..)$')`<br>`REGEXP_MATCHES('ITMO-2024', '[A-Z]+')`<br>`REGEXP_MATCHES('Гитара', '[0-9]+')` | `'{A,BC}'`<br>`'{ITMO}'`<br>`NULL` |
| **REGEXP_REPLACE**| Заменяет подстроку с использованием регулярного выражения. | **Б**<br>**Т**<br>**Г** | `REGEXP_REPLACE('A B', '(.) (.)', '\2 \1')`<br>`REGEXP_REPLACE('Ремарк Э.М.', '([А-Яа-я]+) (.*)', '\2 \1')`<br>`REGEXP_REPLACE('12345', '[a-z]', 'X')` | `'B A'`<br>`'Э.М. Ремарк'`<br>`'12345'` |

### 🎨 Часть 3. Форматирование, регистр и объединение
Функции для склейки слов, приведения текста к красивому виду и сборки предложений.

| Функция       | Описание                                     |           Тип           | Пример выражения                                                                                                   | Результат                                                            |
| :------------ | :------------------------------------------- | :---------------------: | :----------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------- |
| **CONCAT**    | Объединяет две или более строк в одну.       | **Б**<br>**Т**<br>**Г** | `CONCAT('A', 'B', 'C')`<br>`CONCAT('Эрих ', 'Мария ', 'Ремарк')`<br>`CONCAT('ИТМО', NULL, '2024')`                 | `'ABC'`<br>`'Эрих Мария Ремарк'`<br>`'ИТМО2024'`                     |
| **CONCAT_WS** | Объединяет строки с указанным разделителем.  |          **Б**          | `CONCAT_WS(',', 'A', 'B')`                                                                                         | `'A,B'`                                                              |
| **FORMAT**    | Форматирует строку на основе шаблона.        | **Б**<br>**Т**<br>**Г** | `FORMAT('Hello %s', 'PG')`<br>`FORMAT('Гитара: %s, Струн: %s', 'Telecaster', 6)`<br>`FORMAT('Значение: %s', NULL)` | `'Hello PG'`<br>`'Гитара: Telecaster, Струн: 6'`<br>`'Значение: '`   |
| **INITCAP**   | Делает первую букву каждого слова заглавной. | **Б**<br>**Т**<br>**Г** | `INITCAP('hI tHERE')`<br>`INITCAP('превращение франца кафки')`<br>`INITCAP('ИТМО-университет')`                    | `'Hi There'`<br>`'Превращение Франца Кафки'`<br>`'Итмо-Университет'` |
| **LOWER**     | Переводит строку в нижний регистр.           | **Б**<br>**Т**<br>**Г** | `LOWER('ABC')`<br>`LOWER('ТРИУМФАЛЬНАЯ АРКА')`<br>`LOWER('123!@#')`                                                | `'abc'`<br>`'триумфальная арка'`<br>`'123!@#'`                       |
| **UPPER**     | Переводит строку в верхний регистр.          | **Б**<br>**Т**<br>**Г** | `UPPER('abc')`<br>`UPPER('итмо')`<br>`UPPER('   ')`                                                                | `'ABC'`<br>`'ИТМО'`<br>`'   '`                                       |
| **REPEAT**    | Повторяет строку заданное количество раз.    | **Б**<br>**Т**<br>**Г** | `REPEAT('*', 3)`<br>`REPEAT('Дж-дж-дж ', 2)`<br>`REPEAT('Кафка', 0)`                                               | `'***'`<br>`'Дж-дж-дж Дж-дж-дж '`<br>`''`                            |

### 🧹 Часть 4. Очистка, дополнение и кодирование
Функции для выравнивания текста, удаления мусора и работы с кодами символов. ^9f7650

| Функция   | Описание                                                     |           Тип           | Пример выражения                                                                            | Результат                                    |
| :-------- | :----------------------------------------------------------- | :---------------------: | :------------------------------------------------------------------------------------------ | :------------------------------------------- |
| **LPAD**  | Дополняет строку слева указанными символами до нужной длины. | **Б**<br>**Т**<br>**Г** | `LPAD('123', 5, '0')`<br>`LPAD('Том 1', 10, '.')`<br>`LPAD('ИТМО', 2, 'X')`*                | `'00123'`<br>`'.....Том 1'`<br>`'ИТ'`        |
| **RPAD**  | Дополняет строку справа указанными символами.                | **Б**<br>**Т**<br>**Г** | `RPAD('ABC', 6, 'x')`<br>`RPAD('Fender', 10, '-')`<br>`RPAD('Соло', 0, '!')`                | `'ABCxxx'`<br>`'Fender----'`<br>`''`         |
| **LTRIM** | Удаляет указанные символы с начала (слева) строки.           | **Б**<br>**Т**<br>**Г** | `LTRIM('00123', '0')`<br>`LTRIM('***Университет ИТМО', '*')`<br>`LTRIM('Кафка', 'Кф')`      | `'123'`<br>`'Университет ИТМО'`<br>`'афка'`  |
| **RTRIM** | Удаляет указанные символы с конца (справа) строки.           | **Б**<br>**Т**<br>**Г** | `RTRIM('abcxxzx', 'xyz')`<br>`RTRIM('На Западном фронте...', '.')`<br>`RTRIM('123', '456')` | `'abc'`<br>`'На Западном фронте'`<br>`'123'` |
| **TRIM**  | Удаляет символы с обеих сторон (по умолчанию — пробелы).     | **Б**<br>**Т**<br>**Г** | `TRIM(' ABC ')`<br>`TRIM(BOTH '-' FROM '-Gibson-')`<br>`TRIM('')`                           | `'ABC'`<br>`'Gibson'`<br>`''`                |
| **ASCII** | Возвращает ASCII/Unicode код первого символа.                | **Б**<br>**Т**<br>**Г** | `ASCII('A')`<br>`ASCII('К')` *(кириллица)*<br>`ASCII('')`                                   | `65`<br>`1050`<br>`0`                        |
| **CHR**   | Преобразует ASCII/Unicode код обратно в символ.              | **Б**<br>**Т**<br>**Г** | `CHR(65)`<br>`CHR(9834)` *(символ ноты)*<br>`CHR(0)`                                        | `'A'`<br>`'♪'`<br>`Ошибка`                   |
| **MD5**   | Возвращает MD5 хеш строки в шестнадцатеричном виде.          | **Б**<br>**Т**<br>**Г** | `MD5('ABC')`<br>`MD5('Студент ИТМО')`<br>`MD5('')`                                          | `'902fb...'`<br>`'9d4f0...'`<br>`'d41d8...'` |

*\* Если целевая длина в `LPAD`/`RPAD` меньше длины исходной строки, строка будет обрезана.*
